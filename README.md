# 📚 Library Management System

A web-based Library Management System built as our final year project, 2025. 
It allows librarians to manage books, members, issue-return books and borrowing records efficiently.

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS
- **Backend:** PHP
- **Database:** MySQL
- **Email:** PHPMailer

## ✨ Features

- User authentication (Admin & Student login)
- Admin authorised management
- Library card generation
- Add, update, and delete books
- Student registration and profile management
- Issue and return book by scanning qr code from the books
- Issue and return book tracking. Such as statistical data of each user, books, daily report, monthly report and all reports.
- Fine collection procedure
- Fine and book return reminder email notification via PHPMailer
- Book cover uploads

## 🚀 How to Run Locally

1. Clone the repository
   git clone https://github.com/kousikibhukta/library-management-system
2. Import the SQL file into your MySQL database via phpMyAdmin
3. Update database credentials in the connection file
4. Run the project on a local server (XAMPP recommended)
5. Open your browser and go to http://localhost/library-management-system
